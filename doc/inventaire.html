Énumération FamilleObjets

Différents types d'objets possibles.
Valeurs : 
	RIEN,
	POTION, BOMBE,
	CASQUE, TORSE, GANTS, JAMBIERES, BOTTES

Enregistrement TypeObjets

Un ou plusieurs objets identiques.
Champs :
  • famille (FamilleObjets)
  • idObjet (Integer)
  • quantite (Integer)

Procédure initialiserInventaire

Initialise l'inventaire avec des cases vides (famille RIEN).
Aucun paramètre.

Fonction regarderDansInventaire

Indique le contenu d'une case de l'inventaire.
Paramètres :
  • numCase (Integer) : un nombre de 0 à 8, le numéro de la case
Renvoie : un TypeObjets, le contenu de la case

Fonction mettreDansInventaire

Place un nouvel objet dans l’inventaire.
Paramètres :
  • objet (TypeObjets) : le(s) objet(s) à mettre dans l'inventaire
Renvoie : un booléen, VRAI si l’objet à bien été placé ou FAUX si l’objet n’a pas pu être placé dans l’inventaire (plus de place)

Fonction sortirDeInventaire

Récupère un objet depuis l’inventaire (enlève l’objet de l’inventaire et le revoie) Paramètres:
  • numCase Integer : le numéro de la case
  • quantite (Integer) : le nombre d'objets à retirer. Si quantite vaut -1 ou dépasse le nombre d'objets dans la case, tous les objets de la case seront récupérés
Renvoie : un TypeObjets, les objets récupérés
